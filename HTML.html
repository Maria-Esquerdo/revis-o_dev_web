Linguagem de Marcação: HTML

HyperText Markup Language

Estrutura básica de um código HTML com os principais elementos semânticos: 


        <!DOCTYPE html>  -> Define-se o tipo de documento
        <html lang="pt"> -> Define-se o idioma padrão
        <head> -> Aqui temos o cabeçalho, com os metadados.
            <meta charset="UTF-8">  -> Define-se a codificação padrão de caracteres
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>  -> Define-se o título, texto que aparece na aba.
        </head>

        <body> -> Aqui temos o corpo, o que, de fato, aparece no navegador.

            <header>
                Header é o cabeçalho da página HTML.
                Agrupa informações como logo e título do site ou redirecionamentos (nav).
            </header>

            <nav> -> Seção que agrupa redirecionamentos (links).
                <ul>
                    <li><a href="#">Home</a></li>
                    <li><a href="#">Sobre</a></li>
                    <li><a href="#">Contato</a></li>
                    </ul>
            </nav>
            
            <main>
                <section>
                    Divide o conteúdo em blocos temáticos/capítulos. Agrupa elementos que fazem sentido juntos.
                        <article id="bloco1">
                            Envolve conteúdos autossuficientes e independentes
                            Ou seja, conteúdos que fazem sentido sozinhos. Geralmente agrupados por uma seção.
                        </article>

                         <article class="sub_bloco">
                            Conteúdo independente 2
                            Um código pode ter diversos articles e sections, por isso eles podem ser identificados 
                            pelos atributos: 
                            id -> identificador único 
                            class -> identificador múltiplo
                        </article>
                </section>

                <aside>
                    Contém informações periféricas e quase sempre é estilizado, com CSS, para ser uma barra laeral.
                    Geralmente termos relacionados, anúncios, biografias curtas ou widgets.
                </aside>

            </main>

            <footer>
                É o rodapé da página.
            </footer>

        </body>
        </html>
