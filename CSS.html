Linguagem de Estilização CSS

Cascade Style Sheets

Assim como o HTML, o CSS é uma linguagem descritiva, que descreve para o navegador como posicionar cada elemento textual na página web.

Ou seja, as Folhas de Estilo em Cascata são responsáveis pela apresentação visual da página (cores, fontes, espaçamentos, alinhamentos e layouts).

O navegador já tem sua folha de estilo associada padrão, mas é possivel que programadores sobreponham novos estilos.

Podemos estilizar elementos HTML inline, no próprio documento HTML. A estilização inline se sobrepõe às outras, mas não é a forma mais otimizada.

O mais usual é criar um arquivo de estilização CSS e associá-lo através da tag link, no header, ao documento html.

Esse documento CSS tem sua própria sintaxe.

Sintaxe inicial e anexo com o arquivo HTML:

Dentro do arquivo HTML, podemos estilizar de forma inline ou podemos criar uma tag, no head, "style". No entanto, não são formas práticas.

O mais recomendado é criar, separadamente, um arquivo CSS e linkar esse, com a tag link, no head do arquivo HTML.

<link rel="stylesheet" href="nome do arquivo">

Esses arquivos CSS não têm um cabeçalho inicial, como o HTML. Iniciamos diretamente com os seletores e a estilização.


                    seletor 
                    {
                        chave1: valor1;
                        chave2: valor2;
                        chave3: valor3;
                    }

Esses seletores são as tags usadas no HTML. As chaves são os possíveis atributos com seus respectivos valores.

Variações na sintaxe/outras formas de selecionar elementos do HTML:

  • Seletor Universal: *

    Seleciona todos os elementos do arquivo HTML.
    
                                    *{
                                        border: 0px;
                                        margin: 0px;
                                        padding: 0px;
                                    }

  • Seletor por tipo: tipo1, tipo2, tipo3...

    Seleciona os elementos daquele determinado tipo, e, com o uso da vírgula, seleciona mais de um tipo para aquela mesma estilização.
    
                                    a, b, c
                                    {
                                        border: 0px;
                                        margin: 0px;
                                        padding: 0px;
                                    }

  • Seletor de classes: .

    Seleciona todos os elementos pertencentes a uma determinada classe.

    Podemos selecionar elementos que fazem parte de mais de uma classe (classe dentro de classe), também.

    
                                    .classe                 .classeA.classeB  ->Seleciona elementos que fazem parte
                                    {                       {                      da classe A e da classe B.
                                        color: red;             color: blue;
                                    }                       }

  • Seletor de identificador (id): #

    Seleciona todos os elementos pertencentes a um determinado identificador.
    
                                    #id                     #idA#idB  ->Seleciona elementos que fazem parte
                                    {                       {                      do id A e do id B.
                                        color: red;             color: blue;
                                    }                       }

  • Seletor de filhos diretos: >

    Seleciona todos os elementos que são filhos diretos, descendentes diretos, de primeiro grau, do elemento pai.
    
                                    pai > filho  -> Seleciona todos os elementos filhos do elemento pai
                                    {
                                        background-color: blue;
                                    }

                                    Esse seletor pode ser combinado, também: 
                                    a > b > c 
                                    {
                                        Seleciona todos os filhos diretos dos b's que são filhos diretos de a's
                                    }

  • Seletor de filhos (geral):

    É representado por um espaço em branco e seleciona todos os elementos filhos dentro do elemento pai, independente do grau de parentesco.
    
                                    main div
                                    {
                                        Seleciona todas as div's filhas da main, independente de são filhas diretas, netas...
                                    }

  • Seletor de irmão adjacente: +

    Seleciona o elemento adjacente ao especificado, caso seja do tipo dado.

    Exemplo: a + b

    Seleciona o elemento logo depois de a se for do tipo b.

    
                                    h2 + p 
                                    {
                                        Seleciona os parágraafos que estão imediatamente depois de h2 (irmão adjacente).
                                    }

  • Seletor geral de irmãos: ~

    Seleciona todos os elementos do segundo tipo que são irmãos do primeiro tipo. Independente se são irmãos imediatos ou não.

    Exemplo: a ~ b

    Seleciona todos os elementos b que são irmãos de a.

    
                                    h2 ~ p 
                                    {
                                        Seleciona todos os parágrafos irmãos de h2 (abaixo de h2, sendo do mesmo pai).
                                    }

  • Seletor por atributo: []

    Há diversas formas de trabalhar com esse atributo:
    • Seletor simples ([]):
      Seleciona os eleemntos que tenham o determinado atributo ou seleciona os elementos que tenham o determinado atributo com o valor definido.
      
                                        a[href] -> Seleciona todos os elementos a com atributo href.
                                        li[value="5"] -> Seleciona todos os li's que tem atributo value = 5.
    • Para selecionar valores de atributo que começam um com fragmento de texto: ([atributo|="xxx"])
      
                                        a[lang|="pt"] 
                                        {
                                            Seleciona os elementos a's que possuem atributo lang com início pt ou pt-.
                                            Aceita o hífen.
                                        }
    • Para selecionar valores de atributo que começam um com fragmento de texto: ([atributo^="xxx"])
      
                                        a[href|="https"] 
                                        {
                                            Seleciona os elementos a's que possuem atributo href com início exato https.
                                        }
    • Para selecionar valores de atributo que terminam um com fragmento de texto: ([atributo$="xxx"])
      
                                        a[href$="pdf"] 
                                        {
                                            Seleciona os elementos a's que possuem atributo href com final pdf.
                                        }
    • Para selecionar valores de atributo em que um certo fragmento de texto está no meio: ([atributo*="xxx"])
      
                                        a[href*="ufjf"] 
                                        {
                                            Seleciona os elementos a's que possuem atributo href que tenham o fragmento "ufjf" no meio do texto.
                                        }

  • Seletores de pseudoclasses: (:)

    As pseudoclasses adicionam funções extras a elementos do html e não estão contidas previamente nele.
    Estilizam dinamicamente o elemento explicitado sem a necessidade do JavaScript.
    • x:hover ou :active
      O cursor é ativado quando o mouse passa por cima.
    • x:nth-child(3)
      Seleciona os elementos de um a cada três.
    • x:nth-last-of-type(2)
      Seleciona o segudo elemento disponível no elemento pai.

  • Seletor de pseudo-elementos: (::)

    Os pseudo-elementos servem para estilizar uma parte específica do conteúdo de um elemento, diferente das pseudo-classes, que estilizam o elemento inteiro.
    Exemplos:
    • ::before
      Adiciona algo logo antes de um elemento.
    • ::after
      Adiciona algo logo depois de um elemento.
    • ::first-line
      Altera algo na primeira linha.

Herança e características herdadas:

Nem todas as propriedades CSS são herdadas automaticamente.

Propriedades de texto, geralmente são herdadas por padrão, mas propriedades de layout, caixa e posicionamento não são.

Podemos controlar a herança com algumas palavras-chave:

  • inherit
    Força um elemento filho a herdar o estilo do pai.
  • initial
    Cancela qualquer herança e redefine a propriedade para o valor padrão de fábrica do navegador.
  • unset
    Funciona como um "reset". Se a propriedade for herdada por natureza, ela age como inherit. Se não for, age como initial.
